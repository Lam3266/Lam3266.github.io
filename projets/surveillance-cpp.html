C++ · Structures de données · INF3105

Système de surveillance
de localisation

Application C++ modulaire de surveillance de localisation avec historique horodaté. Implémente des structures de données avancées (arbre AVL, arbre map, pile) pour gérer efficacement les entrées, sorties et l'historique des entités suivies.

⎇ Voir le code sur GitHub
C++ Arbre AVL Arbre Map Makefile Tests unitaires

Structures de données utilisées

  • arbreavl.h — arbre AVL auto-équilibré pour indexation rapide
  • arbremap.h — arbre map pour association clé-valeur
  • pile.h — pile générique pour gestion d'états
  • historique.h/.cpp — enregistrement horodaté des événements
  • dateheure.h/.cpp — type date/heure avec opérations de comparaison

Points techniques clés

  • Implémentation manuelle d'arbres AVL avec rotations simples et doubles
  • Gestion d'états de présence avec transitions horodatées
  • Suite de tests unitaires avec cas limites (entrées invalides, séquences vides)
  • Code modulaire séparant structures de données, logique métier et I/O

Architecture modulaire

  • Séparation stricte structures de données / logique métier / entrées-sorties
  • Templates C++ pour structures génériques réutilisables (pile, arbre)
  • Interfaces bien définies entre les modules — couplage minimal
  • Gestion explicite de la mémoire et des ressources (RAII)
  • Suite de tests couvrant les cas limites : séquences vides, doublons, grandes entrées
