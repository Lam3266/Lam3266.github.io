Suivi du rythme cardiaque
en temps réel

Système de mesure et visualisation du rythme cardiaque combinant un capteur analogique Arduino, un serveur Flask et une interface web. Les données BPM sont transmises en série depuis l'Arduino, traitées par Python et affichées en temps réel dans le navigateur.

⎇ Voir le code sur GitHub ▶ Vidéo de démonstration
Arduino Uno SEN-11574 Python Flask HTML / JS Transmission série

Fonctionnement

  • Le capteur SEN-11574 lit le signal analogique du pouls
  • L'Arduino calcule le BPM à partir des intervalles entre pics
  • Les données sont transmises via liaison série USB au serveur Python
  • Flask expose un endpoint SSE (Server-Sent Events) pour le frontend
  • L'interface web affiche le BPM en temps réel avec un graphique dynamique

Points techniques clés

  • Lecture et traitement du signal analogique brut sur Arduino
  • Communication série fiable entre microcontrôleur et serveur Python
  • Architecture Flask légère avec streaming de données vers le navigateur
  • Validation du calcul BPM comparé au moniteur série Arduino
