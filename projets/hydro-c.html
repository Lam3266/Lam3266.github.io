Simulation de production
hydroélectrique

Programme C simulant la production d'énergie hydroélectrique à partir de données de débit et de hauteur de chute. Inclut validation complète des entrées, calculs physiques de puissance et structuration modulaire du code.

⎇ Voir le code sur GitLab UQAM
C GitLab Makefile

Points techniques clés

  • Validation rigoureuse des données d'entrée (débits, dimensions, paramètres physiques)
  • Calculs de puissance hydroélectrique avec formules physiques
  • Structuration modulaire : séparation validation / calcul / affichage
  • Gestion des cas limites et des erreurs d'entrée utilisateur

Logique algorithmique

  • Validation des paramètres physiques (débit, hauteur de chute, rendement) avant tout calcul
  • Calcul de puissance : P = ρ × g × Q × H × η — avec constantes et précision numérique
  • Gestion des cas limites : valeurs nulles, valeurs négatives, dépassements de capacité
  • Architecture pipeline : lecture → validation → calcul → affichage résultat
  • Séparation des responsabilités : pas de logique de calcul dans les fonctions d'I/O
