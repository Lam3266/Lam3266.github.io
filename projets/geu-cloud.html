Projet principal · SaaS

GEU Cloud

Plateforme SaaS de gestion d'équipements industriels et techniques — conçue pour centraliser l'inventaire, les tickets de maintenance, la traçabilité et les tableaux de bord en un seul outil. Projet en développement avec une architecture API REST découplée du frontend React.

Privé — Code source non public

Problème résolu

La gestion d'équipements dans les organisations repose souvent sur des tableurs dispersés, sans traçabilité des interventions ni historique des pannes. GEU Cloud centralise ces données avec des rôles utilisateurs, une API REST structurée et des alertes de maintenance.

Stack technique

Django REST Framework React TypeScript PostgreSQL API REST JWT Auth QR Code

Modules principaux

  • Inventaire d'équipements avec historique complet
  • Système de tickets de maintenance (création, assignation, clôture)
  • Traçabilité par QR Code — scan pour accéder à la fiche équipement
  • Gestion des emplacements et des sites
  • Rôles utilisateurs hiérarchisés (admin, technicien, lecteur)
  • Tableaux de bord avec indicateurs de maintenance
  • API REST documentée pour intégrations tierces

Architecture générale

Backend Django REST découplé du frontend React/TypeScript. Authentification par JWT. Base de données PostgreSQL avec modèles relationnels (équipements, tickets, emplacements, utilisateurs). Permissions gérées au niveau de l'API. Déploiement prévu sur un VPS avec Nginx.

Défis techniques

  • Conception d'un système de permissions granulaires par rôle et par ressource
  • Génération et validation des QR Codes liés aux équipements
  • Gestion des états du cycle de vie des tickets (ouvert → en cours → résolu → clôturé)
  • Sérialisation imbriquée des relations Django REST sans N+1 queries
